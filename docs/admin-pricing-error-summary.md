# 后台价格页访问报错摘要

## 页面地址

- `http://localhost:5173/admin/pricing`

## 当前现象

- 页面无法打开
- 浏览器显示“无法访问此站点”
- `localhost` 拒绝建立连接
- 错误码：`ERR_CONNECTION_REFUSED`

## 报错页主要文字

### 标题

- 无法访问此站点

### 主提示

- `localhost` 拒绝建立连接

### 建议项

- 检查网络连接
- 检查代理、防火墙和 DNS 配置
- 如果使用代理服务器，检查系统网络设置

## 说明

这个报错表示当前本地开发服务没有成功对外提供 `http://localhost:5173/admin/pricing` 页面，因此浏览器无法连上本地服务端口。

它和普通页面内脚本报错不同，这一页属于“连接失败”页面，说明浏览器在请求页面时，本地服务本身不可达。

## 排查方向

1. 确认本地 Vite 开发服务是否已启动
2. 确认 `5173` 端口是否被正确监听
3. 确认服务是否因依赖解析错误、配置错误或启动失败而退出
4. 确认没有被防火墙、代理或系统策略拦截

## 本次上下文中的关联问题

- 后台价格页此前出现过 `dayjs` 相关模块解析报错
- 修复过程中多次需要重启本地开发服务
- 当前这个页面截图对应的是“服务未连通”状态，而不是页面渲染中的 JS 报错状态
